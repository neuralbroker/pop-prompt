# PopPrompt

A clean, modern, lightweight and fast prompt saver for Windows.

## Features

- Create, edit, and delete prompts
- Star your favorites
- Organize with tags
- Search and filter prompts
- Copy to clipboard with one click
- Dark/Light theme toggle
- Auto-save with indicator

## Quick Start

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Keyboard Shortcuts

- `Ctrl+N` - New prompt
- `Ctrl+F` - Search
- `Ctrl+Shift+C` - Copy to clipboard
- `Delete` - Delete selected prompt
- `Escape` - Clear search / Close dialog
